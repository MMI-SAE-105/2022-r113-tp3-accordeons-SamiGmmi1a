Accordéon avec x-collapse

Le plugin JavaScript collapse est utilisé pour afficher et masquer le contenu. Les boutons ou les ancres sont utilisés comme déclencheurs mappés à des éléments spécifiques que vous basculez. Réduire un élément animera la height de sa valeur actuelle à 0 . Étant donné la façon dont CSS gère les animations, vous ne pouvez pas utiliser de padding sur un élément .collapse . À la place, utilisez la classe comme élément d'emballage indépendant.
